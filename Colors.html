There are 4 ways to define a color

Just saying the name of the color.
There are 140 supported colors for HTML



RGBA format first 3 going 0-255
and the last value representing opacity going 0-1



HEX in format #RRGGBB where each 2 pairs go 00-ff



HSLA in format 0-360 representing a wheel Red - 0, Green - 120, Blue - 240 for first part, 0-100% representing the saturation of the color for the second, 0-100% light for the amount of white in the color for third and final of 0-1 alpha.
