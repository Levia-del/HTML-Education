Link element, href attribute represents the link itself

In order to control how the link is going to open use the target attribute

_self is set on default, will open it in the same window

_blank will open it in a new window

_parent will open it in the first window of the path that got you to this window

_top will open it in the same way as _self but breaking out of all sub frames or windows


Absolute vs Relative URL



Absolute URL - a full web adress

Relative URL - a shorted adress within the same website, without the "http://www"

Any element can be a link, even an image



Use mailto: in href attribute to use an email adress as a link

To use button as a link, in the onclick attribute type document.location='adress'



Use the title attribute to edit what is displayed when the link is hovered over
